# Election Mentor AI 🗳️

**Election Mentor AI** is an interactive, responsive web application designed to guide and educate voters through the election process. Built with premium design tokens, dark-mode glassmorphism, and a dynamic flow, the app provides a first-class voting onboarding and simulation experience.

## ✨ Features
- **Intelligent Onboarding**: Multi-step flow for voter age, location, and type selection.
- **Interactive Voting Simulation**: Step-by-step 3D-visual walkthrough of entering a booth, ID verification, and EVM voting.
- **Real-time Timeline**: Detailed timeline of election events with calendar integration.
- **My Progress Tracking**: High-fidelity cards to view verification status and election readiness.
- **Learn Resources**: Rich resources regarding voting eligibility, process, and required documents.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS, JS
- **Icons**: FontAwesome 6
- **Typography**: Google Fonts (Inter)

## 📂 Project Structure
- `index.html` - Dashboard & AI chat
- `age-selection.html` - Onboarding: Age Check
- `location-selection.html` - Onboarding: Location Details
- `voter-type.html` - Onboarding: Voter Status
- `timeline.html` - Election Timeline
- `simulation.html` - Simulation Mode Step 1: Booth Entry
- `id-verification.html` - Simulation Mode Step 2: ID Verification
- `evm-voting.html` - Simulation Mode Step 3: EVM Voting
- `ink-marking.html` - Simulation Mode Step 4: Ink Marking
- `journey-progress.html` - Voter Registration Status tracker
- `learn.html` - Education & Help Hub
- `styles.css` - Global design system and premium layout stylings
